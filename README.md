# Overview

Shows that gifs can be displayed in SDL2.

IntroScene.cpp contains very ugly concept of proof.

# Build

Makefile

# Implementation

This program is written in SDL2, on Linux Mint.

Uses the following SDL2 libraries:
 - SDL2
 - SDL2_image
 - SDL2_ttf
 - SDL2_mixer
 - libgif
